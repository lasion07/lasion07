Các đặc trưng cơ bản của python:

1.Ngôn ngữ lập trình đơn giản, dễ học
    các cú pháp trong python rất ngắn gọn, đơn giản và dễ đọc. Cho phép lập trình viên tập trung vào giải pháp hơn là cú pháp.

2.Miễn phí, mã nguồn mở:
    Có thể tự do sử dụng và phân phối Python hoàn toàn miễn phí. Ngoài ra, còn có thể tùy ý thay đổi mã nguồn của nó. Trên trang mạng có rất nhiều cộng đồng học tập và sử dụng python nên có thể học hỏi, tìm hiểu.

3.Khả năng di động, đa nền tảng:
    Có thể viết mã và chạy Python trên các nền tảng khác nhau như Windows, Mac, Linux mà không cần viết lại mã hay chỉnh sửa. Tuy nhiên các hệ thống có một số tính năng khác nhau nên một số mã nguồn phải phụ thuộc vào chúng.

4.Khả năng mở rộng và nhúng:
    Bạn có thể dễ dàng sử dụng và kết hợp với các ngôn ngữ khác như C, C++,... vào trong code python. Điều này cung cấp khả năng scripting tốt mà các ngôn ngữ lập trình khác khó có thể làm được.

5.Ngôn ngữ thông dịch cấp cao.
    Python có cơ chế tự động chuyển đổi mã nguồn khi chạy thành bytecode nên việc cần làm là chạy đoạn code mà không cần lo lắng về quản lý bộ nhớ, liên kết các thư viện,...

6.Thư viện tiêu chuẩn lớn
    Python có một lượng lớn thư viện tiêu chuẩn giúp cho làm dự án, viết chương trình trở nên đơn giản hơn nhiều vì không phải tự viết tất cả mã code mà đem lại hiệu năng tốt.

7.Hướng đối tượng
    Python là ngôn ngữ lập trình hướng đối tượng. Python hỗ trợ kế thừa và với OOP có thể xử lý các vấn đề phức tạp bằng cách chia ra các vấn đề nhỏ.
